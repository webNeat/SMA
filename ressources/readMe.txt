Simulation des relations entre les étudiants et les entreprises:
    Une entreprise:
        utilise/demande des compétences
        faire de la publicité ( diffusion d'information )
        recruter des gens pour stage ou emploi.
    Les etudiants:
        a un réseau de connaissances
        a des competences
        des objectifs (veut apprendre des choses )
        collecter les informations sur les entreprises qui les interessent. 
        partager les informations
        faire des demande de stage ou emploi
        devenir employés dans une entreprise
        se réunir et construir une nouvelle entreprise
Fonctionalités:
    Les interfaces:
        Acceuil:
            Menu:
                - Continuer => Simulation
                - Nouvelle simulation
                - Configuration 
        Nouvelle Simulation:
            Menu:
                - Ajouter Entreprise
                - Ajouter Ecole
                - Commencer la simulation => Simulation
        Ajouter Entreprise:
            Formulaire:
                nom
                groupe des compétences
                nombre d'employés
                nombre moyen de stage offerts par an
                position (x,y)
        Ajouter Ecole:
            Formulaire:
                nom
                position (x,y)
                entreprises partenaires
                Pour chaque niveau
                    groupe des compétences
                    nombre d'étudiants
                    pourcentage de réussite
                    durée et période du stage ( s'il y en a)
        Configuration:
            Menu:
                - Edition des groupes des compétences et compétences
                - Modifier Statistiques
